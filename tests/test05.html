MAYA

MAYA - A classic audiocard featuring superb compatibility for your home studio. Unlike most regular soundcards, MAYA was designed with the musician in mind. Today's recording/sequencing applications and software synthesizers are extremely powerful. With soft synths and effects you can easily create a complete studio environment inside your PC. But with a typical soundcard you will have many problems. In most cases the latency is so high that you can't use software synths efficiently. Also, most regular soundcards only provide average audio quality; microphone inputs, for example, are often not usable for serious music production.

MAYA is different: the card was especially designed to work with any current music and audio application. The card comes with excellent driver support (ASIO 2.0, DirectSound, MME) - that allows usage with virtually any popular application and yields ultra-low latency: 3ms at 44.1kHz using Steinberg's ASIO driver, and an exceptional 1.5ms using Cakewalk's SONAR. This is the simple solution for playing software synthesizers in realtime. Employing 20-bit DAC (digital-to-analog converters), 18-bit ADC (analog-to-digital converters) as well as RCA-connectors, the audio quality for a card in this price range is excellent. What this means is noise won't be an issue when using MAYA. A special feature is the included microphone preamp with phantom power.

MAYA also provides an optical digital output (S/PDIF, TOS-Link). It is possible to output signals with 44.1 and 48kHz sample rates. This allows you to transfer signals to MD or DAT-recorders without any signal loss.

All functions of the card can be controlled with the convenient Control Panel. The usage of the card is very simple and you can learn all functions within minutes.


Audio applications can access the card using Audiotrak's proprietary E-WDM driver which contains MME, DirectSound and ASIO 2.0 drivers. While MME is the default driver model used for soundcards under Windows, you can use DirectSound for soft synths (like Rebirth), media players, as well as games. The DirectSound driver works with very low latency so you can play signals without any audible delay. With the ASIO 2.0 driver you can use the card with professional audio/MIDI sequencers like Steinberg's Cubase VST and Emagic Logic. You can even play virtual instruments (VSTi/ DXi) and use effect plugins (VST/DirectX) with only 3ms latency (ASIO 2.0 at 44.1kHz) or a mere 1.5ms using Cakewalk's SONAR. Even many of the more expensive audiocards cannot achieve such low latency. More and more audio applications are employing ASIO 2.0 (which can be accessed from the E-WDM driver) such as the audio program Reason from Propellerheads. If you want to run SONAR simultaneously with Nemesys' GigaStudio, it is possible with Maya's E-WDM driver (GigaStudio version 2.20).

You can use MAYA for several purposes: monitoring/recording software synthesizers, high-quality recordings from line or microphone signals, or to transfer data to your MD/DAT recorder. No matter which application you have for MAYA, you can be sure that the card works with all common music programs. To ensure that, we regularly test the card with popular applications. Of course, you can also contact us when you require any advice using MAYA.

Specifications

2 In and Output channels (analog) using RCA connectors

In and Output level: -10dBv (unbalanced), +9.0dBv max.

18bit ADC (90dB real dynamic range, A-weighted)

20bit DAC (90dB real dynamic range, A-weighted)

Supported sample rates: 32, 44.1, 48kHz

20bit S/PDIF digital output (optical TOS-Link): 44.1 & 48kHz

Microphone amplifier: +45dB pre-set; +20dB switchable; 0-18dB adjustable; 12V phantom power switchable

Internal input for CD-Audio signals

Full-duplex (simultaneous recording and playback)

E-WDM drivers for Windows 98 SE, Windows ME, Wind
